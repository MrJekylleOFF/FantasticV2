# mrw_chest

item safe, with password

[INFORMATION]

[EN]
safe in item, with password (positions, password, inventory stored in a json file). 
You have 3 tries to try to unlock the chest otherwise it hangs for 2 minutes. 
You can change the time, the test number in the client as well as the maximum of characters allow for the password



[FR]
coffre fort en item, avec mot de passe ( positions, mot de passe, inventaire stockés dans un fichier json) .
Vous avez 3 essais pour essayer de déverouiller le coffre sinon il se bloque pendant 2 minutes .
Vous pouvez changer le delais, le nomber d'essais dans le client ainsi que le maximum de caractéres autoriser pour le mot de passe

[REQUIEREMENTS]

- Es_extended

[INSTAL]

1) clone the repertories
2) start resource 
3) enjoy

[BUG]

vous devez vous deconnecter/reconnecter pour acceder au coffre, si vous venez de le poser

you must disconnect / reconnect to access the safe, if you have just put it
