# fivem-bejesus
Be Jesus for the afternoon.

# Requirements
- FiveM/CitizenFX
- No framework requirements.

# Installation

* Extract the folders into your `resources` directory.
* Add `start fivem-bejesus` in your server-cfg.
* Check the config.lua for controls.
